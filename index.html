Matplotlib is another Python library. This library is used for data visualization, supporting what Pandas cannot. Matplotlib is widely used and allows for the support of images and graphs a like. This project is specifically using the pyplot interface, which is for data graphics specifically.

What is Biopics.csv?

Most important to this project is the dataset itself. This dataset was pulled from FiveThirtyEight's GitHub page. FiveThirtyEight is an ABCNews-owned website that utilizes data to increase public knowledge surrounding everything from popular culture to politics. This dataset specifically was used to develop the article "Straight Outta Compton Is The Rare Biopic Not About White Dudes" in 2015. Data was collected from IMDb, the Internet Movie Database.

Now that these columns have been dropped, we need to fill in the missing data. Data is missing in two columns: box_office and subject_race. The first column accounts for the amount of money the film made at the box office. The missing data in this column will be replaced with "0". The column 'subject_race' indicates the race of the person the biopic is about. To fill in missing data, data from the 'person_of_color' column will be used. If the data in that column is coded '0', then the 'subject_race' column will be filled in with the string "White". The following code sections show this process.

In[258]:
def convert(value):
    value = value.replace('$', '').replace(',', '')
    if 'M' in value:
        value = float(value.replace('M', '')) * 1_000_000  
    elif 'K' in value:
        value = float(value.replace('K', '')) * 1_000 
    elif '-' in value:
        value = float(value.replace('-','0'))
    else:
        value = float(value)
    return value
df['box_office'] = df['box_office'].apply(convert)
df.loc[df['subject_race'].isna() & (df['person_of_color'] == 0), 'subject_race'] = 'White'
df['year_release'] = pd.to_numeric(df['year_release'], errors='coerce')
